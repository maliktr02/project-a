PHYSICS FORMULAS FOR PROJECT A
-------------------------------------------------------------------------------

FREE_FALL:
->      vy(new) = vy(old) + g * dt
->      y(new) = y(old) + vy(new) * dt
-------------------------------------------------------------------------------

DISTANCE_VECTOR:
->      d = sqrt[(x2 - x1)^2 + (y2 - y1)^2]
-------------------------------------------------------------------------------

COLLISION_DETECTION:
->      d <= r1 + r2
-------------------------------------------------------------------------------

OVERLAP_RESOLUTION:
->      dd = (r1 + r2) - d
-------------------------------------------------------------------------------

ADJUSTMENT_AMOUNT:
->      x1(new) = x1 - [(x2 - x1 / d) * (dd / 2)]
->      y1(new) = y1 - [(y2 - y1 / d) * (dd / 2)]
-------------------------------------------------------------------------------

MERGE_CENTER:
->      x(average) = (x1 + x2) / 2
->      y(average) = (y1 + y2) / 2
-------------------------------------------------------------------------------

ELASTIC_BOUNCE:
->      vy(new) = -e * vy(old)
-------------------------------------------------------------------------------

WALL_COLLISION:
->      x - r <= 0, x = r (left wall)
->      x + r >= W, x = W - r (right wall)
->      y + r >= H, y = H - r (bottom wall)
-------------------------------------------------------------------------------

EXPLANATIONS:
->      e = damping coefficient, e ~ 0.5 to 0.8
->      g = acceleration due to gravity, g = 9.8 m/s^2 or g = 10 m/s^2
->      d = distance between two objects
->      vx = velocity in x direction
->      vy = velocity in y direction
->      x = position in x direction
->      y = position in y direction
->      r = radius of object
->      x(average) = average position in x direction
->      y(average) = average position in y direction
->      W = width of box
->      H = height of box
-------------------------------------------------------------------------------
