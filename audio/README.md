# Audio Files

Place your audio files here in WAV format.

## Structure
- audio/intro.wav - Intro screen sound
- audio/sfx/drop.wav - Object drop sound
- audio/sfx/merge.wav - Merge sound (or merge_1.wav through merge_12.wav for per-level)
- audio/sfx/click.wav - Button click
- audio/sfx/combo.wav - Combo sound
- audio/sfx/danger.wav - Danger warning
- audio/sfx/gameover.wav - Game over
- audio/sfx/taunt.wav - Taunt sound
- audio/music/bgm.wav - Background music (loops)

If files are missing, procedural audio fallbacks will be used.
